<h1>プロジェクト</h1>
<p>アーカイブHPをVue.jsで変更する</p>


<h1>環境</h1>

<p>Node 12.13.0</p>
<p>npm 6.12.0</p>

<p>Vue 2.6.10</p>
<p>Nuxt 2.0.0</p>
<p>Vuetify 2.1.9</p>

<h1>Start</h1>

<p> Npm(Node Package Manager)が必要です </p>
  
<p>$ npm run install</p>

<p>$ npm run dev</p>
